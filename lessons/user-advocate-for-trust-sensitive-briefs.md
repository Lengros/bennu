# User Advocate for Trust-Sensitive Briefs
date: 2026-03-19
scope: delegation
rule: Trust/consent screens where user anxiety dominates require user_advocate competencies (emotional journey, trust signals) in the brief.

## Why
Curator writing a trust-sensitive brief directly produced a flat, mechanical result. Synthesizing the brief-writer with the user_advocate catalog pattern — which adds emotional journey mapping and psychology-grounded anti-patterns — produced a materially better result in the same session.

## How to apply
When the task involves a user-facing flow where anxiety, consent, or trust is the primary constraint: add user_advocate competencies to the Specialist brief (emotional journey, trust signals, anti-patterns from user psychology). Do not write such briefs directly at Curator level.
