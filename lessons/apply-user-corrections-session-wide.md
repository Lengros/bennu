# Apply User Corrections Session-Wide
date: 2026-04-04
scope: process
rule: A process correction applies to all remaining tasks of that type; third violation: stop, name the pattern, recalibrate, confirm with user.

## Why
A user correction about process ("always delegate this type of work") was applied to the current task but not internalized for the rest of the session. The same mistake recurred twice more before explicit recalibration happened.

## How to apply
When the user corrects a process: state the generalized rule out loud, apply it to all pending tasks of the same type in this session, not just the one in front of you. If the same correction is violated a third time: stop, name the pattern, ask the user to confirm the rule is understood before continuing.
