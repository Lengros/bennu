# No-Run Discipline: Cast a Subagent for Domain Artifacts
date: 2026-06-06
scope: delegation
rule: Cast a per-task subagent for any domain artifact; inline only for meta-edits and one-line factual fixes.

## Why
In sessions without active enforcement hooks, the only barrier to role collapse is behavioral. The temptation to write/edit a domain artifact "because it's faster / hooks aren't watching" is the collapse signal — it leads to the same quality degradation as skipping delegation in a managed run.

## How to apply
Collapse signal: about to Write/Edit a domain artifact directly "because it's faster." Correct response: cast a per-task subagent (Agent tool) with proper framing instead. Inline remains valid for: meta-configuration files, state/retro directories, and literal one-line factual corrections.

## Disposition
Kept for artifact-shaped work; embody legitimized for synthesis-shaped output only per Bennu §3.3 (task-shape rule, user-approved 2026-06-10). The legacy no-run lane mechanics are dropped; the behavioral principle survives.
