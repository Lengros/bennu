## Roster — named starting points the gate can propose

Not full packages yet — distinct directions to riff at Stage 2, each clearly *not* the
default attractor. Mix, remix, or invent beyond these.

- **swiss / international** — strict grid, huge Helvetica-ish type, flat ink-on-paper, red accent, zero ornament. Authority via restraint.
- **brutalist web** — raw HTML energy, mono type, hard 1px borders, system-default-but-on-purpose, high contrast, no shadows.
- **editorial luxe** — oversized serif display, generous whitespace, ink + one rich jewel tone, magazine columns. (NB: serif ≠ Fraunces-default — go Tiempos/Canela-grade scale and confidence.)
- **neo-brutalist pop** — thick black borders, hard offset shadows, saturated flat blocks, chunky type. Playful, loud, memorable.
- **glass + grain** — frosted translucent panels over a deep gradient, fine noise texture, soft inner light. (Distinct from default soft-rounded SaaS — commit to depth + grain, not bubbles.)
- **terminal / phosphor** — black bg, monospace everything, green/amber CRT glow, scanlines, blinking caret. Hacker/ops register.
- **dark luxe / fintech** — near-black + a single metallic accent (gold/chrome), tight type, precise hairlines, restrained motion. Premium, serious.
- **risograph / print** — 2–3 spot inks, registration-offset overlap, halftone texture, paper warmth. Crafty, human, analog.
- **maximal gradient** — bold duotone/mesh gradients used with intent (not the generic purple-blue SaaS hero), large type knocked out of color fields.
- **blueprint / technical** — cyan lines on dark navy, drafting-grid, annotation callouts, dimension markers. Engineering-credible.

When proposing, give each chosen direction a palette (tokens + roles), a display/body type
pair, a **signature mechanic** (the one unmistakable move), and an explicit ban-list — per
the gate spec in `SKILL.md` §2.
