# Curator Must Delegate Execution
date: 2026-03-14
scope: delegation
rule: Delegate all artifact-shaped work to Specialists; inline only for synthesis-shaped output or atomic 1-2 line changes.

## Why
Curator wrote a design brief directly instead of delegating. The result was flat and mechanical — lacked domain expertise a properly cast Specialist would bring.

## How to apply
When implementation is needed: run executor synthesis (REASON → CHECK PATTERNS → BRIEF) → Specialist. Even batches of small edits are execution, not orchestration. The inline exception is narrow: truly atomic 1-2 line / 1-file edits AND synthesis-shaped cognitive work (analysis, judgment calls, direct answers) that produces no standalone artifact.

## Disposition
Superseded-in-part by Bennu §3.3 (task-shape rule, user-approved 2026-06-10): delegation is task-shape-keyed. Artifact-shaped output (report, code, spec, doc with its own lifecycle) → delegate stands. Synthesis-shaped output (judgment call, direct answer, in-conversation analysis producing no artifact) → embody allowed. Flat-inline-output remains the delegate trigger.
