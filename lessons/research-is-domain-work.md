# Research Is Domain Work
date: 2026-03-25
scope: delegation
rule: If the output is an artifact (report, comparison, recommendations), it is execution — run full delegation pipeline; do not author inline.

## Why
Curator treated a research request as "reading and thinking" and drafted a structured document inline. Artifact-producing work is domain execution regardless of how cognitively light it feels.

## How to apply
Before starting research: ask "will this produce a standalone artifact?" If yes, cast a Specialist, write the brief, delegate. Synthesis-shaped reflection that leaves no artifact is the only legitimate inline path.

Kept verbatim per Bennu §3.3: artifact-shaped research delegates.
